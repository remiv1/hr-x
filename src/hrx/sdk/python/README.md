# HRX Python SDK (hrxlib)

Paquet Python fournissant les modèles et helpers pour le format HRX.

Installation (locally):

```bash
pip install build
cd src/hrx/sdk/python
python -m build
pip install dist/hrxlib-0.1.0-py3-none-any.whl
```

Usage:

```python
from hrx import Candidate
print(Candidate)
```
